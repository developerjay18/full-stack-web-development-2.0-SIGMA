HTML Tags :-

  1. "HTML":- Defines the root of an HTML document. The opening and closing tags of an HTML document.
  2. "HEAD":- Contains metadata about the HTML document. Contains information about the document such as title, scripts, and styles.
  3. "TITLE":- Defines the title of the document displayed in the browser. Displays the title of the page in the browser's title bar.
  4. "BODY":- Contains the visible content of the document. Contains the main content of the web page that's displayed on the browser.
  5. "DIV":- Defines a container for HTML elements. A versatile tag used to group HTML elements and apply CSS styles.
  6. "P":- Defines a paragraph of text. Indicates a block of text and creates a space between paragraphs.
  7. "IMG":- Inserts an image into an HTML document.
  8. "A tag":- Creates a hyperlink to another webpage or file.
  9. "UL":- Defines an unordered list.
  10. "TABLE":- This tag is used to make tables into our webpage.It includes many other tags into it like "TR", "TH", or "TD".
