Most common Formatting tags

  1. "h1 - h6"
    • Use "h1" for the main heading of a page, and use subsequent headings for subheadings in a hierarchical order.
    • Use headings to create a clear and concise outline of the content of the page.
    • Use headings to emphasize important points or sections of the page.
    • Don't use heading tags purely for styling purposes. Use CSS to style headings instead.
    • Use only one "h1" tag per page, as it should represent the main topic of the page.

  2. "p" (paragraph)
    • Use "p" to enclose paragraphs of text on a web page. Each "p" tag should contain a single paragraph
    • Don't use "p" tags for headings or subheadings. Use the appropriate heading tags for that purpose.
    • Use semantic and descriptive text for the content of each paragraph, and avoid using filler or meaningless text.
    • Break up long paragraphs into shorter ones for better readability.
    • Use proper indentation and spacing to make the content of each paragraph more readable.

  3. "sub" (subscript)
    • Don't use "sub" for text that is merely smaller in size. Use CSS to control the size of text instead.
    • Use "sub" for superscript text that is positioned higher than the surrounding text.
    • Use semantic and descriptive text for the content of subscript text.
    • Use proper line-height and font-size to ensure the subscript text is legible and not too small.
    • Avoid overusing subscript text, as it can make the content difficult to read and understand.

  4. "sup" (superscript)
    • Use "sup" to indicate superscript text. Superscript is typically used for footnotes, endnotes, mathematical exponents,
      or text that is smaller and positioned higher than the surrounding text.
    • Don't use "sup" for text that is merely smaller in size. Use CSS to control the size of text instead.
    • Use proper line-height and font-size to ensure the superscript text is legible and not too small.
    • Avoid overusing superscript text, as it can make the content difficult to read and understand.
    • Use CSS to style superscript text instead of adding inline styles to individual "sup" tags.

  5. "strong"
    • Use "strong" to indicate text that is important, significant, or requires emphasis. This can include keywords, important phrases, or warnings.
    • Don't use "strong" for text that is merely bolded for stylistic purposes. Use CSS to control the styling of text instead.
    • Use semantic and descriptive text for the content of "strong" tags.
    • Avoid overusing "strong" tags, as it can make the content appear cluttered and distract from the most important points.
    • Use CSS to style "strong" text instead of adding inline styles to individual "strong" tags.

  6. "strike"
    • Use the "s" or "del" tag to indicate text that has been deleted or removed.
    • Use the "ins" tag to indicate text that has been added or inserted.
    • Don't use the "strike" tag, as it is deprecated and may not be supported by all web browsers.
    • Use CSS to style deleted or inserted text instead of relying on the default styling of the tags.
    • Avoid overusing these tags, as it can make the content appear cluttered and difficult to read.
